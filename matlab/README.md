# BSDF Matlab/Octave implementation


## Installation

Download `bsdf.m` and place it in a directory where Matlab can find it, e.g.
by doing:

```matlab
addpath('/path/to/bsdf');
```

## Development

Run `invoke -l` in this directory for available tasks (like tests).
